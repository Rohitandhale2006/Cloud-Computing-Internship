# Task 1 - Cloud Storage Setup

## Objective
Create and configure cloud storage using Google Cloud Platform.

## Services Used
- Google Cloud Storage
- Google Cloud Console

## Steps Performed
1. Created a Google Cloud Project.
2. Created a Cloud Storage Bucket.
3. Uploaded image files to the bucket.
4. Configured object permissions.
5. Generated public URL for uploaded objects.

## Deliverables
- Storage Bucket Created
- Files Uploaded Successfully
- Public Access Configured

## Outcome
Successfully implemented cloud storage and file hosting using Google Cloud Storage.
